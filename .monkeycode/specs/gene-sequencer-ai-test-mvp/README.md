# 基因测序仪 AI 测试条目与验证方案生成工具 MVP 文档包

本目录包含基因测序仪 AI 测试条目与验证方案生成工具的 MVP 设计文档，用于立项评审、研发启动和验收准备。

## 文档列表

| 文档 | 内容 |
|---|---|
| `requirements.md` | MVP 需求规格说明书 |
| `architecture.md` | 系统架构设计 |
| `data-api.md` | 数据模型与核心 API 设计 |
| `ai-prompt-schema.md` | AI 编排与 Prompt/Schema 设计 |
| `implementation-plan.md` | MVP 实施计划与里程碑 |
| `acceptance-test-plan.md` | 验收测试方案与评估指标 |

## 当前确认的关键口径

1. 系统采用统一资料池，资料可由用户上传，系统智能识别标签和去重，管理员审核后发布。
2. 项目通过标签、权限和适用范围引用资料，不按项目独立重复上传资料。
3. 子系统分类目录 V1 采用一级系统域加二级子系统结构。
4. 验证方案模板 V1 采用用户提供的 Word 示例结构。
5. 测试方案类文档按 `3.x` 测试项目拆分为测试条目资产。
6. 拆分条目需要分别映射测试对象、测试层级、测试类型、主子系统、关联子系统和分类依据。
7. 系统需要建立测试归口资产，用于表达某类对象或某类变更应组合哪些测试条目。
8. Jira 和 DFMEA 作为风险知识源，用于补充回归测试和风险驱动测试建议。
9. 本地知识库和测试资产库优先，联网大模型作为可选外部参考。
10. 最终输出为可编辑、可校验、可导出的 Word 验证方案。
